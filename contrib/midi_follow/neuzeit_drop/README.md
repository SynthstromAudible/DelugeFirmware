# MIDI Follow Mode templates for Neuzeit Instruments' Drop MIDI Control Center

Contains a project file for the Neuzeit Instruments' [Drop MIDI Control Center](https://www.neuzeit-instruments.com/Drop)

This project assumes each track has been set to the came channel number. (e.g. Track 6 is MIDI channel 6)
This project also assumes the Drop is running firmware version 2.01 or later (in order to use eight layers)

- Layer 1 is a mixer layer controlling the first eight tracks of the song
- Layers 2-8 are details controls for each track
- Layer 2 controlling track one is configured for a kit track
- Layers 3-8 controlling tracks 2-7 are conifgured for synths

Thanks to James Macaulay for the [Drop Editor](https://jamesmacaulay.github.io/dropedit/) to allow easy copying parameters and settings between layers.
