# Note / Drum keyboard view

#OFFICIAL #COMMUNITY #KEYBOARD #SYNTH #MIDI #CV

```shortcut
press(KEY)
```

For synth, MIDI, and CV clips this opens keyboard view. For kits it shows drum pads. You can still change presets and parameters while in keyboard view.

# Change Keyboard Layout

#COMMUNITY #KEYBOARD #SYNTH #MIDI #CV

```shortcut
hold(KEY) + turn(SELECT)
```

Changes keyboard mode between Isomorphic, In-Key, Piano, Chord, Chord Library, and Norns. Note: Chord, Chord Library and Norns need to be enabled in Community Features.

# Chord Keyboard Row / Column Mode

#COMMUNITY #KEYBOARD #SYNTH #MIDI #CV

```shortcut
hold(SHIFT) + turn(X)
```

Switches between row mode and column mode in the chord keyboard.

# Scroll Chord Keyboard Horizontally

#COMMUNITY #KEYBOARD #SYNTH #MIDI #CV

```shortcut
turn(X)
```

Moves across scale degrees in the chord keyboard.

# Scroll Chord Keyboard Vertically

#COMMUNITY #KEYBOARD #SYNTH #MIDI #CV

```shortcut
turn(Y)
```

Moves vertically through scale degrees in the chord keyboard.

# Chord Library Voicing

#COMMUNITY #KEYBOARD #SYNTH #MIDI #CV

```shortcut
hold(GRID) + turn(X)
```

Changes voicing for all chords in the selected row.

# Keyboard scale / root
