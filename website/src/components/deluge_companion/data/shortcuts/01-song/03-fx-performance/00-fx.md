# Enter Performance View

#COMMUNITY #PERFORMANCE #ARRANGER #SESSION

```shortcut
press(PERFORMANCE)
```

Opens Performance View from Session or Arranger View when tapping the keyboard button. The short press time is configurable in the settings menu under "hold press time."

# Enter Performance View (Temporarily)

#COMMUNITY #PERFORMANCE #ARRANGER #SESSION

```shortcut
hold(PERFORMANCE)
```

Opens Performance View Temporarily from Session or Arranger View while holding the keyboard button and switches back to Session or Arranger View after releasing the keyboard button.

# Performance View Editing Mode

#COMMUNITY #PERFORMANCE

```shortcut
hold(SHIFT) + press(PERFORMANCE)
```

Opens the value and parameter editing modes for Performance View.

# Save Performance View Changes

#COMMUNITY #PERFORMANCE

```shortcut
press(SAVE) + press(PERFORMANCE)
```

Saves edited Performance View values.

# Reload Performance View Changes

#COMMUNITY #PERFORMANCE

```shortcut
press(LOAD) + press(PERFORMANCE)
```

Reloads the saved Performance View layout.

# Clear Performance View Held Values

#COMMUNITY #PERFORMANCE

```shortcut
hold(X) + press(BACK)
```

Clears held FX values in Performance View.

# Master Compressor Mode

#COMMUNITY #PERFORMANCE #ARRANGER #SESSION

```shortcut
press(ENTIRE), turn(UPPER_PARAM)
```

When Affect Entire is on and Sidechain is enabled, this adjusts the master compressor mode.

# VU Meter

#COMMUNITY #PERFORMANCE #ARRANGER #SESSION

```shortcut
press(ENTIRE), press(ParamButton1)
```

Press the LEVEL/PAN BUTTON while it is already selected to show the VU meter.
