# Resample / record all audio output (begin instantly)

#OFFICIAL #ARRANGER #SESSION #CLIP

```shortcut
hold(SHIFT) + press(RECORD)
```

Pressing shift+record begins output-recording instantly.

Records or resamples all audio output. Press Record again to stop.

The Deluge can record its audio output to a file on the SD card. This is useful both as a sonic tool, where the user may create interesting sounds on the Deluge and “resample” them for further use, or to allow the user to make high-quality recordings of compositions or performances.

This is not to be confused, though, with live looping, one use of which allows the Deluge’s output to be recorded into an audio clip.

Recorded WAV files will appear in the SAMPLES/RESAMPLE folder of the SD card. Also, after you've made a recording in this way, the next time you are taken to the file browser, you will be defaulted to your new file, allowing you to easily add it to a kit or as an audio clip.

# Resample / record all audio output (begin with playback)

#OFFICIAL #ARRANGER #SESSION #CLIP

```shortcut
hold(RECORD) + press(PLAY)
```

To begin output-recording at the exact same instant that playback is begun, press record+play.

# Loop resample / record all audio output (begin with playback, stop at end of the loop)

#OFFICIAL #ARRANGER #SESSION #CLIP

```shortcut
hold(RECORD) + press(PLAY), hold(RECORD) + press(PLAY)
```

Repeat Record + Play before the end of the loop. If already playing, hold Record + Play to quantize recording to the loop start.

# Resample / record all audio output (begin with voice)

#OFFICIAL #SYNTH #KIT #MIDI #CV #KEYBOARD

```shortcut
hold(RECORD) + press(AUDITION)
```

To begin output-recording at the exact same instant that a voice is sounded, hold down record and press an audition pad (or one of the main pads if you're in keyboard view).

Press Record to stop.

# Record sample into a new kit

#OFFICIAL #KIT

```shortcut
hold(SHIFT) + press(KIT), press(RECORD)
```

Better for offline recording than using audio clips or live looper recording.

Create a new, blank kit clip by pressing shift+kit (provided you were in clip view already). A new kit is created, and the user is placed in the file browser. However, to record a new sound rather than select a file, simply press the record button at this stage. The sound recorder is entered, and recording begins immediately, indicated by the word “REC” blinking on the Deluge’s display.

Press the record button again to end recording.

Note: After recording samples into a kit, the Deluge digitally normalizes the volume level of the recorded sample to make it as loud as possible without clipping.

# Record sample into an existing kit

#OFFICIAL #KIT

```shortcut
hold(AUDITION) + press(RECORD)
```

To record further samples into an existing kit, hold down the “audition” pad on an unoccupied row, and press the record button. The sound recorder will be re-entered, and an additional sample may be recorded. This process may be repeated as many times as desired.

The sound recorder may also be entered via the sound editor for a particular sound.
