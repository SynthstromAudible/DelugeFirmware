# Launch clip - queue

#SONG

```shortcut
press(LAUNCH)
```

Queues the clip to stop or start at the next loop.

# Launch clip - immediately

#SONG

```shortcut
hold(SHIFT) + press(LAUNCH)
```

Stops or starts the clip immediately.

# Fast scroll song list

#SONG

```shortcut
press(LOAD), hold(SHIFT) + turn(SELECT)
```

# Create clip (empty row)

#SONG

```shortcut
press(GRID_UNLIT)
```

Creates the clip and enters clip view. You can create unlimited clips. Scroll up or down in song mode to view more rows.

# Enter clip (non-empty row)

#SONG

```shortcut
press(GRID)
```

Enters the clip to view or edit it. Return to song mode with the Song button.

# Move row clip up or down

#SONG

```shortcut
hold(GRID) + turn(Y)
```

# Clip section color - change or create new

#SONG

```shortcut
hold(SHIFT) + press(AUDITION)
```

Cycles through existing section colors. If the end is reached, a new color is created. Maximum of 12 sections / colors.

# Launch section

#SONG

```shortcut
press(AUDITION)
```

All audition / section rows of the same color flash, then launch together when the current longest-playing clip loops or starts again.

# Section repeat (hold audition for 1 sec)

#SONG

```shortcut
hold(AUDITION), turn(SELECT)
```

Changes INFInite to a repeat count for the section. The display counts down; press Select to cancel the section switch during countdown.

# Clone/duplicate clip (hold source, press destination)

#SONG

```shortcut
hold(GRID) + press(GRID)
```

Destination row can be another existing clip. The cloned clip is inserted.

# Delete clip

#SONG

```shortcut
hold(GRID) + press(SAVE)
```

Cannot undo delete.

# Solo clip - queue

#SONG

```shortcut
hold(X) + press(LAUNCH)
```

The soloed clip turns blue and other mute pads dim. You can solo more than one clip. Press Mute/Launch to unsolo.

# Solo clip - immediately

#SONG

```shortcut
hold(SHIFT) + hold(X) + press(LAUNCH)
```

Immediately solos the clip. You can solo more than one clip.

# Clip parameter change (per clip)

#SONG

```shortcut
hold(GRID) + turn(PARAMETER)
```

# Clip parameter change (per song)

#SONG

```shortcut
press(ENTIRE), turn(PARAMETER)
```

# Change existing clip preset

#SONG

```shortcut
hold(GRID) + turn(SELECT)
```

# Change existing clip to synth

#SONG

```shortcut
hold(GRID) + press(SYNTH)
```

On an empty clip, pressing SYNTH also changes it to a synth clip.

# Change existing clip to MIDI

#SONG

```shortcut
hold(GRID) + press(MIDI)
```

On an empty clip, pressing MIDI also changes it to a MIDI clip.

# Change existing clip to CV

#SONG

```shortcut
hold(GRID) + press(CV)
```

On an empty clip, pressing CV also changes it to a CV clip.

# Create audio track (in empty clip)

#SONG

```shortcut
hold(GRID) + press(SELECT)
```

# Check name and type of non-empty clip

#SONG

```shortcut
hold(GRID)
```

The LED flashes with the clip name and the clip-type LED lights up, for example MIDI.

# Clip Settings Menu (Song Grid View)

#SONG

```shortcut
hold(GRID) + press(SELECT)
```

# Clip Settings Menu (Song Row View)

#SONG

```shortcut
hold(LAUNCH) + press(SELECT)
```

Uses status pad (mute/launch) in first column of sidebar.

# Row colour from song row view

#SONG

```shortcut
hold(GRID) + hold(SHIFT) + turn(Y)
```

In Song Row view, cycle through colors for the held pad row.

# Song views

#SONG

```shortcut
hold(SONG) + turn(SELECT)
```

Changes song view between rows and grid.
