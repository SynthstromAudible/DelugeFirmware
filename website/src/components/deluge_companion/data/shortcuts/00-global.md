# View zoom level

#GLOBAL

```shortcut
press(X)
```

# Change zoom level

#GLOBAL

```shortcut
hold(X) + turn(X)
```

When you `turn(X)` clockwise, the resolution doubles. So instead of 16th notes, you'll see 32nd notes displayed.

Similarly, when you `turn(X)` counterclockwise, the resolution is halved.

Note: White notes in song and clip mode require zooming in to see exact position in the timeline.

# Scroll horizontal

#GLOBAL

```shortcut
turn(X)
```

Scroll left and right across the grid.

# Scroll vertical

#GLOBAL

```shortcut
turn(Y)
```

Scroll up and down across the grid.

# Undo

#GLOBAL

```shortcut
press(BACK)
```

Note: If a preset is changed or you enter sound editor, then the undo history is cleared.

# Redo

#GLOBAL

```shortcut
hold(SHIFT) + press(BACK)
```

Note: If a preset is changed or you enter sound editor, then the redo history is cleared.

# New song

#GLOBAL

```shortcut
hold(SHIFT) + press(LOAD)
```

Creates a new song.

# Delete song

#GLOBAL

```shortcut
press(LOAD), turn(SELECT), hold(SHIFT) + press(SAVE), press(SAVE)
```

Scroll to the song to delete, then use Shift and Save/Delete to delete it.

# Load song

#GLOBAL

```shortcut
press(LOAD), turn(SELECT), press(LOAD)
```

Note #1: For gapless play, hold press(LOAD) to delay launch if large samples are included.

Note #2: Hold SHIFT and turn SELECT to fast-scroll songs on the display.

# Save song

#GLOBAL

```shortcut
press(SAVE), turn(SELECT), press(SELECT)
```

Note: Song slots with a '.' after the name already exist, and 'OVERwrite' will flash when trying to save. A, B, C designate iterations of the same song.

# Change tempo (course)

#GLOBAL

```shortcut
turn(TEMPO)
```

Press and turn for 1 BPM increments. This is configurable in community feature settings.

# Change tempo (fine)

#GLOBAL

```shortcut
hold(TEMPO) + turn(TEMPO)
```

Fine tempo changes use smaller increments than the coarse tempo shortcut.

# Change tempo (secret power up)

#GLOBAL

```shortcut
hold(LEARN) + turn(TEMPO)
```

Use this to increase tempo from 0 to FAST, FAST!

# Metronome

#GLOBAL

```shortcut
hold(SHIFT) + press(TAP)
```

Default metronome level is 50%.

# Swing

#GLOBAL

```shortcut
hold(SHIFT) + turn(TEMPO)
```

Swing interval is 1/16th notes by default, but can be adjusted in the settings menu.

# File system up

#GLOBAL

```shortcut
press(BACK)
```

Go back up one level from within the file system.

# Adjust brightness

#GLOBAL

```shortcut
hold(SHIFT) + hold(LEARN) + turn(Y)
```

# Settings menu

#GLOBAL

```shortcut
hold(SHIFT) + press(SELECT)
```

# Save song, collect all samples

#GLOBAL

```shortcut
press(SAVE), turn(SELECT), hold(SELECT), press(SELECT)
```

Saves the song, creates a folder with the same name, and copies all used sample files into it under /SONGS.

# Pad refresh rate

#GLOBAL

```shortcut
hold(CLIP) + turn(SELECT)
```

Changes the pad refresh rate, which can help when video recording the pads.

# Start playback from current scroll position (Alternative Playback)

#GLOBAL

```shortcut
press(PLAY)
```

Requires "Alternative Playback Start Behaviour" enabled. With playback off.

# Start playback from start (Alternative Playback)

#GLOBAL

```shortcut
hold(X) + press(PLAY)
```

Requires "Alternative Playback Start Behaviour" enabled. With playback off.
