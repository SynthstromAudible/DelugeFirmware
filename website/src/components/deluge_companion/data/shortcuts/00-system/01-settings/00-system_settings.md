# Enter Settings menu

#OFFICIAL #MENU_SETTINGS

```shortcut
hold(SHIFT) + press(SELECT)
```

# Adjust brightness

#OFFICIAL

```shortcut
hold(SHIFT) + hold(LEARN) + turn(Y)
```

# Pad refresh rate

#OFFICIAL

```shortcut
hold(CLIP) + turn(SELECT)
```

Changes the pad refresh rate, which can help when video recording the pads.

# Metronome

#OFFICIAL

```shortcut
hold(SHIFT) + press(TAP)
```

Default metronome level is 50%.
