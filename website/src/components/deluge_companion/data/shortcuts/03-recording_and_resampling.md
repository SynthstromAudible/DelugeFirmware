# Resample / record all audio output

#KIT

```shortcut
hold(SHIFT) + press(RECORD)
```

Records or resamples all audio output. Press Record again to stop.

# Loop resample

#KIT

```shortcut
hold(RECORD) + press(PLAY), hold(RECORD) + press(PLAY)
```

Repeat Record + Play before the end of the loop. If already playing, hold Record + Play to quantize recording to the loop start.

# Record sample into a kit

#KIT

```shortcut
hold(SHIFT) + press(KIT), press(RECORD)
```

Better for offline recording than using audio clips or live looper recording.

# Load sample into a kit row

#KIT

```shortcut
hold(AUDITION) + press(LOAD), turn(SELECT), press(SELECT)
```

Scroll to a sample and press Select. Shift + Browse/Grid shortcut on an existing row can also open the browser.

# Load sample into a kit row silently

#KIT

```shortcut
hold(SHIFT) + hold(AUDITION) + press(LOAD), turn(SELECT), press(SELECT)
```

# Synth resample

#KIT

```shortcut
hold(RECORD) + press(AUDITION), hold(RECORD) + press(GRID)
```

Press Record to stop.

# Load all samples from folder to kit

#KIT

```shortcut
hold(SHIFT) + press(KIT), turn(SELECT), hold(SELECT), menu(NONE), press(SELECT)
```

Scroll to the parent folder of the samples you want, or to a sample within the folder, then choose ALL.

Beware of CPU and memory overhead when loading large amounts of samples into kits. Keep KIT files minimal for best performance. This does not apply to multi-sampled synths.

# Slice sample

#KIT

```shortcut
hold(SHIFT), press(KIT), turn(SELECT), hold(SELECT), menu(NONE), press(SELECT), menu(NONE), press(SELECT)
```

Scroll to a sample, choose SLICE, then select the number of slices from 2-256.

You can add more samples to sliced kits, but not more slices to existing kits. Only one sliced sample per kit is possible.

# Rename kit row or instrument

#KIT

```shortcut
hold(AUDITION) + press(GRID)
```

After recording samples into a kit, the Deluge digitally normalizes the volume level of the recorded sample to make it as loud as possible without clipping. Recorded audio clips are not normalized this way.
