This page will provide descriptions of and links to various Deluge Applications that have been built to enhance the Deluge experience.

## Deluge Companion (Shortcut Browser) <Badge text="Web" variant="tip" />

A website for easy browsing and searching of shortcuts for the Synthstrom Deluge.

Application Page: https://fheft.github.io/deluge-companion/

Source Code: https://github.com/fheft/deluge-companion

## DEx: Deluge EXtensions - See Your Deluge Like Never Before <Badge text="Web" variant="tip" />

Unlock the full potential of your Synthstrom Deluge with DEx! A web-based tool offering display mirroring, debugging, and advanced control via WebMIDI. Try the live demo!

![DEx mobile display](./assets/DEX_mobile.png)

Application Page: https://dex.silicak.es

Source Code: https://github.com/silicakes/deluge-extensions

## Delugr (File Browser) <Badge text="Web" variant="tip" />

Delugr is a web-app that lets you browse the contents of your Deluge memory card.

It uses the new File System Access API to read the contents of your memory card, so there's no need to install anything! As of December 2024, the API is not yet supported by Firefox, Safari or any of the mobile browsers.

Application Page: https://delugr.haila.fi

Source Code: https://github.com/MrHaila/delugr

Synthstrom Forum Page: https://forums.synthstrom.com/discussion/4547/delugr-a-wip-deluge-file-browser-in-the-browser

## Downrush - Waverly (WAV File Editor) <Badge text="Web" variant="tip" />

Waverly is a program for editing .WAV files using an old version of WaveSurfer

Application Page: https://www.fentonia.com/deluge/waverly/

## Downrush - ViewScore (Project Editor) <Badge text="Web" variant="tip" />

ViewScore is a program for viewing Deluge XML files such as KIT, SYNTH, and SONG. downrush stuff runs mostly, but has not been updated to show new Community Edition features.

Application Page: https://www.fentonia.com/deluge/viewScore/

Source Code: https://github.com/SynthstromAudible/DelugeWeb/tree/main/downrush/viewScore

## Midian (MIDI File to XML File Converter) <Badge text="Web" variant="tip" />

This program converts Midi-format files into Synthstrom Deluge SONG.XML files. This beta version also allows you to convert SONG.XML files into Midi-format files.

Application Page: https://www.fentonia.com/catnip/midian/index.html

## Deluge Commander <Badge text="Desktop" variant="tip" />

Utility to organize Synthstrom Deluge sample library, by moving and renaming samples while updating references in KITS, SONGS and SYNTHS.

Application Page: https://neuma.studio/deluge-commander/

Source Code: https://github.com/drbourbon/deluge-commander

## Deluge Preset Builder <Badge text="Web" variant="tip" />

A web application used to build kits, multisamples and presets for the synthstrom deluge.

Application Page: https://deluge-kit-generator.vercel.app

## Dealleague (XML Fixer) <Badge text="Desktop" variant="tip" />

Program to analyse and fix missing samples.

It can also set entire volumes in kits and synths and Osc volumes at once to avoid loud mistakes when jamming with others.

With this newest release you can create textfile lists of kits, synth and folders.

Download Link and Source Code: https://github.com/Queroland/DelLeague/releases/tag/V2.0

Synthstrom Forum Page: https://forums.synthstrom.com/discussion/4247/deluge-xml-fixer-and-with-set-volumes/p1?new=1
