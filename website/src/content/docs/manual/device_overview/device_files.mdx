This page provides an overview of the various device files that are saved on the SD card and contain settings which are loaded automatically or manually by a user using the File Browser.

Some files have a fixed name and file path while others can be renamed and moved around by the user within specified folders based on the file type.

Files saved on the SD card include:

- MIDIFollow.xml
- PerformanceView.xml
- CommunityFeatures.xml
- Song project XML files
- Synth preset XML files
- Kit preset XML files
- MIDI presets XML files
- MIDI device definition XML files
- Pattern XML files
