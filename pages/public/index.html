Community Firmware

Synthstrom Audible released the source code of the Deluge firmware and started a community project inviting everyone to participate in improving and extending the Deluge user experience.

Following are all important links to get started with using and testing the firmware as well as participating in the project thorugh feedback, ideas, suggestions and adding to the source code

While the project aims to never break compatibility or lose user data, things can break during development so please make sure you have a recent backup of your SD card before using the provided firmware files. You can always go back to the official firmware but be aware that songs created with community might not be fully compatible with official firmware.

We hope you enjoy using the community firmware as much as we enjoy working on it!
